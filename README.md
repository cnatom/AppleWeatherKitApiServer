## 使用

### 1.下载本项目

```
git clone https://github.com/cnatom/ByteDanceCampusWeatherServer.git
```

### 2.导入依赖包

```
pip install -r requirements.txt
```

### 3.启动服务
运行`app.py`文件

### 4.调试
